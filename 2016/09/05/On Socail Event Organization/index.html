Social Event Organization

Problem Definition

there is a set of users U who must be assigned to events or activities from a given set of possible events A

we define the utility of this assignment as:

$$\mu(S,a)=(1-\alpha)\sum_{u\in S}\sigma_{u,a}+\alpha\sum_{u,v\in S,u\not =v}w(u,v)$$

an assignment is a partial function \(M:U\to A\)

for \(a\in A\), we denote by \(M^{-1}(a)\) the (possibly empty) subset of users assign to a:

$$M^{-1}(a)=\lbrace u \in U\;|\;M(u) = a\rbrace$$

we define the social welfare of a feasible assignment M as:

$$\omega (M)=\sum_{a\in A}\mu (M^{-1}(a),a)$$

our objective is to find a feasible assignment that maximizes the overall social welfare
